# demo
前后端分离的毕设项目-轮廓模式算法展示系统； 
其中demo为后端代码，show为前端代码；
前端采用vue+elementui+echarts实现；
后端使用springboot实现；
分为用户管理和算法运行两大部分；
需要自取，方便的话也可点个收藏哈！！！
